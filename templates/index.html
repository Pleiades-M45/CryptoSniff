Frequently Asked Questions

Everything you need to know about CryptoSniff's cryptojacking detection capabilities.

What is cryptojacking and how does it work?

+

Cryptojacking is the unauthorized use of someone else's computer to mine cryptocurrency. Attackers secretly install mining software or run mining scripts in web browsers, using your device's processing power to generate cryptocurrency for their benefit while slowing down your system and increasing energy costs.

How accurate is CryptoSniff's detection?

+

CryptoSniff achieves 98% accuracy using advanced machine learning algorithms trained on thousands of system behavior patterns. Our Random Forest model analyzes 14 different system metrics including CPU usage, network activity, and I/O operations to identify cryptojacking activities with high precision.

What detection methods does CryptoSniff offer?

+

We offer two primary detection methods: CSV file analysis for batch processing of system logs, and real-time form-based analysis where you can manually input system metrics. Both methods use the same AI-powered detection engine for consistent and reliable results.

Is my data safe when using CryptoSniff?

+

Absolutely! CryptoSniff processes all data locally and doesn't store or transmit your system information to external servers. Your uploaded CSV files and manually entered data are analyzed in real-time and then discarded, ensuring complete privacy and security.

What should I do if cryptojacking is detected?

+

If CryptoSniff detects cryptojacking activity, immediately run a full antivirus scan, check for suspicious browser extensions, monitor running processes, and consider changing passwords for accounts accessed on the affected device. For persistent issues, professional IT support may be required.

Can CryptoSniff prevent cryptojacking attacks?

+

CryptoSniff is primarily a detection tool that identifies ongoing cryptojacking activities. While it doesn't prevent attacks directly, early detection allows for quick response and mitigation. For prevention, we recommend using ad blockers, keeping systems updated, and implementing comprehensive security solutions.
